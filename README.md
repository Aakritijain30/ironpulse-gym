## ✨ Features

- ✅ Fully Responsive (Mobile, Tablet, Desktop)
- ✅ Smooth Scroll Navigation
- ✅ Animated Stats Counter
- ✅ Scroll Reveal Animations
- ✅ Mobile Hamburger Menu
- ✅ Contact Form with Validation
- ✅ Toast Notifications
- ✅ Pricing Plans Section
- ✅ Services Grid
- ✅ Dark Theme with Orange Accents

---

## 🛠️ Tech Stack

| Technology | Use |
|---|---|
| **HTML5** | Page structure & content |
| **CSS3** | Styling, animations, responsive layout |
| **JavaScript (Vanilla)** | Interactivity, scroll effects, form validation |

## 📄 Sections

1. **Home / Hero** — Bold headline with CTA buttons and stats bar
2. **About** — Gym intro with feature highlights
3. **Services** — 6 service cards (Weight Training, Cardio, HIIT, Spinning, Martial Arts, Yoga)
4. **Pricing** — 3 membership plans (Basic $29 / Premium $59 / Elite $99)
5. **Contact** — Contact info + working contact form
6. **Footer** — Logo and copyright
